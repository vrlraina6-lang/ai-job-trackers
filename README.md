# AI Job Application Tracker

Built for Gappy AI Hackathon 2026 using Lemma SDK.

## Problem
Students lose track of job applications and miss follow-ups.

## Solution
AI Job Tracker with Lemma SDK:
- Tracks applications (company, role, status, date)
- AI Strategist drafts follow-up messages by urgency
- Add jobs in plain text — AI auto-parses
- Flags overdue follow-ups automatically
